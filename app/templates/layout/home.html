Bower In Your Django Projects.

Bower is an easy to use FrontEnd Package manager, even though its popularity it's not very used in Django projects due the need of installing Node JS if the projects is being built on the server among other reasons.

View Sample Project on GitHub

Adding Dependencies

In your settings.py file describe the dependencies you want to use.

# Project dependencies
BOWER_INSTALLED_APPS = (
    'jquery#1.9',
    'underscore',
    'bootstrap',
)

Installing Dependencies

From the command line

./manage.py bower install

Including dependencies using Pipeline

From the command line

PIPELINE_CSS = {
    
    # Libraries
    'libraries': {
        'source_filenames': (   
            'bower_components/bootstrap/dist/css/bootstrap.css',
        ),
        'output_filename': 'css/libs.min.css',
    }
}

Jenkins Deploy

In your jenkins.sh build file, make sure that bower components get fetched befor statics are collected.

#...

# Install bower 
./manage.py bower install

# Collect static files
./manage.py collectstatic --noinput --ignore=*.sass

#...

.gitignore configuration

As bower components get fetch on build, we don't need to have them in our repository.

# Ignore bower components
/app/static/bower_components/
